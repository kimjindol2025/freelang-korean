## [1.0.0] - 2026-03-27

### 🎉 공식 릴리스

#### Added
- ✅ 한글 문법 v1.0 (47개 키워드)
  - P0: 변수, 반환, 만약, 함수, 반복, 아니면, 타입, 상수 (8개)
  - P1: 사용, 가져오기, 비동기, 대위, 패턴, 열거형 (6개)
  - P2: 특성, 구현, 제네릭, 모듈 (4개)
  - P3: 뮤텍스, 세마포어, 채널, 선택, 블록, 잠금, 해제, 신호 (8개)
  - 기타: 구조체, 함수, 공집합, 참거짓, 문자열, 숫자, 배열, 맵 등 (21개)

- ✅ K-StdLib 16개 모듈
  - **crypto/**: OTP, AES-256, ARIA, SEED, 인코딩
  - **compliance/**: PIPA, RRN, BRN, Phone 검증
  - **isms/**: 감사 로그, RBAC, 의심 활동 감지
  - **proof/**: SHA-256 해시, 증명 아키텍처
  - **stdlibs/**: array, string, math, json, date, io, object

- ✅ 비동기 프로그래밍
  - async/await 문법
  - Promise 기반 구현
  - 에러 처리 (try-catch)

- ✅ 동시성 프로그래밍
  - Mutex (상호배제)
  - Channel (채널)
  - Semaphore (세마포어)
  - Select (선택)

- ✅ 패턴 매칭
  - 리터럴 매칭
  - 범위 매칭
  - 와일드카드 매칭
  - OR 패턴
  - 가드 조건

- ✅ 자가호스팅 (Self-Hosting)
  - K-FreeLang으로 K-FreeLang 컴파일러 구현
  - 완전 독립형 (TypeScript 의존성 제거)
  - 바이너리 무결성 검증

- ✅ 성능 최적화

  - 라벨 62% 감소

## 버전 관리 정책

### Semantic Versioning
- **MAJOR**: 호환되지 않는 변경 (문법 변경, 키워드 제거 등)
- **MINOR**: 하위 호환 기능 추가 (새 키워드, 새 모듈 등)
- **PATCH**: 버그 수정 (성능 개선, 컴파일 오류 수정 등)
