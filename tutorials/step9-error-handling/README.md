## 📚 핵심 개념

### try/catch 구문
```freelang
시도 {
  // 위험한 코드
} 잡기 (에러) {
  // 에러 처리
}
```

### 에러 타입
- **RuntimeError**: 실행 중 발생
- **TypeError**: 타입 불일치
- **ValueError**: 값 오류
- **ZeroDivisionError**: 0으로 나누기

### 디버깅 기법
```freelang
출력("디버그: 변수값 = " + 변수)  // 중간 상태 확인
```

## 📝 예제

### 예제 1: try/catch
```freelang
시도 {
  변수 결과 = 10 / 0  // 에러!
} 잡기 (에러) {
  출력("에러 발생: 0으로 나눌 수 없습니다")
}
```

### 예제 2: 입력값 검증
```freelang
함수 나이_검증(나이) {
  시도 {
    만약 나이 < 0 {
      반환 거짓
    }
    반환 참
  } 잡기 (에러) {
    출력("에러: " + 에러)
  }
}
```

### 예제 3: 파일 처리
```freelang
시도 {
  // 파일 열기
} 잡기 (에러) {
  출력("파일을 열 수 없습니다")
}
```
