## 📚 핵심 개념

### 구조체 정의
```freelang
구조체 사용자 {
  이름: 문자열
  나이: 숫자
  이메일: 문자열
}
```

### 구조체 인스턴스 생성
```freelang
변수 kim = 사용자 {
  이름: "김철수",
  나이: 30,
  이메일: "kim@example.com"
}
```

### 필드 접근
```freelang
kim.이름      // "김철수"
kim.나이      // 30
kim.이메일    // "kim@example.com"
```

## 📝 예제

### 예제 1: 기본 구조체
```freelang
구조체 학생 {
  이름: 문자열
  학번: 숫자
  학점: 실수
}

변수 lee = 학생 {
  이름: "이순신",
  학번: 20210101,
  학점: 4.0
}

출력(lee.이름)
출력(lee.학번)
```

### 예제 2: 구조체와 함수
```freelang
구조체 좌표 {
  x: 숫자
  y: 숫자
}

함수 거리_계산(point1, point2) {
  // 두 점 사이의 거리 계산
}
```

### 예제 3: 구조체 배열
```freelang
변수 책들 = [
  책 { 제목: "Python 기초", 저자: "김철수", 가격: 35000 },
  책 { 제목: "JavaScript 완전정복", 저자: "이순신", 가격: 45000 }
]
```
