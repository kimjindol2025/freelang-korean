# 🎓 K-FreeLang 10단계 학습 코스

**완벽한 한국어 프로그래밍 언어를 배워보세요!**

> K-FreeLang은 한국 개발자를 위한 완전한 독립 프로그래밍 언어입니다.
> 이 코스를 통해 초급부터 고급까지 단계별로 배울 수 있습니다.

---

## 📚 학습 구조

### **Tier 1: 기초 (Step 1-3)** - 2시간
한글 변수명과 기본 문법을 배웁니다.

### **Tier 2: 중급 (Step 4-7)** - 3시간
함수, 배열, 객체를 다루는 실전 프로그래밍을 배웁니다.

### **Tier 3: 고급 (Step 8-10)** - 2.5시간
모듈, 에러 처리, 웹 프로그래밍을 배웁니다.

---

## 🎯 10단계 커리큘럼

### **Step 1️⃣: 안녕, 프리랭!** (초급 | 20분)
- K-FreeLang 설치 및 환경 설정
- 첫 코드 작성
- 한글 출력

**학습 파일**: `step1-hello-korean/`

### **Step 2️⃣: 변수와 자료형** (초급 | 30분)
- 변수 선언 (한글 변수명)
- 기본 자료형 (숫자, 문자열, 참거짓)
- 연산자 (산술, 비교, 논리)

**학습 파일**: `step2-variables-types/`

### **Step 3️⃣: 조건문과 반복문** (초급 | 40분)
- if/else 조건문
- for/while 반복문
- break/continue 제어

**학습 파일**: `step3-control-flow/`

### **Step 4️⃣: 함수 정의와 호출** (중급 | 40분)
- 함수 선언 문법
- 매개변수와 반환값
- 재귀 함수

**학습 파일**: `step4-functions/`

### **Step 5️⃣: 배열 다루기** (중급 | 35분)
- 배열 생성과 접근
- 배열 메서드 (push, pop, filter, map)
- 다중 배열

**학습 파일**: `step5-arrays/`

### **Step 6️⃣: 객체와 구조체** (중급 | 40분)
- 객체 생성과 속성
- 구조체 정의
- 메서드 추가

**학습 파일**: `step6-objects/`

### **Step 7️⃣: 문자열 조작과 정규식** (중급 | 45분)
- 문자열 메서드
- 문자열 템플릿
- 정규식 기본

**학습 파일**: `step7-strings/`

### **Step 8️⃣: 모듈과 라이브러리** (고급 | 40분)
- 모듈 import/export
- 표준 라이브러리 사용
- 외부 패키지 활용

**학습 파일**: `step8-modules/`

### **Step 9️⃣: 에러 처리와 디버깅** (고급 | 35분)
- try/catch 예외 처리
- 에러 타입과 메시지
- 디버깅 기법

**학습 파일**: `step9-error-handling/`

### **Step 🔟: 실전 프로젝트** (고급 | 1시간)
- 계산기 만들기
- TODO 앱 만들기
- 간단한 웹 스크래핑

**학습 파일**: `step10-projects/`

## 🚀 시작하기

### 설치
```bash
npm install -g kfreelang
```

### 첫 코드 실행
```bash
cd tutorials/step1-hello-korean
kfreelang 1-hello.free
```

### 학습 순서
1. **Step 1 시작**: `step1-hello-korean/README.md` 읽기
2. **예제 코드 실행**: `*.free` 파일 실행해보기
3. **연습 문제 풀기**: `exercises.md`의 문제 풀이
4. **솔루션 비교**: `solution.free`와 비교하기
5. **다음 Step으로**: Step 2로 넘어가기

## 🎁 보너스 자료

### 한글 문법 치트시트
```freelang
// 변수 선언
변수 이름 = "값"

// 함수 정의
함수 함수명(매개변수) {
  반환 결과
}

// 조건문
만약 조건 {
  코드
}

// 반복문
반복 (변수 in 배열) {
  코드
}
```

### 추천 학습 경로
- **완전 초보**: Step 1 → 2 → 3 → 4 → 10 (간단한 프로젝트)
- **프로그래밍 경험 있음**: Step 1 → 2 → skip 3 → 4 → 5 → 6 → 10
- **빠른 학습**: Step 1 → 2 → 4 → 5 → 10 (핵심만)
- **깊이 있는 학습**: Step 1-10 모두 (권장)
