# Step 1️⃣: 안녕, 프리랭!

**난이도**: ⭐ 초급 | **소요 시간**: 20분

> K-FreeLang의 첫 번째 단계입니다.
> 설치부터 첫 코드 실행까지 모든 과정을 배웁니다.

---

## 🎯 학습 목표

이 Step을 완료하면 다음을 할 수 있습니다:
- ✅ K-FreeLang 설치하기
- ✅ 첫 번째 프로그램 작성하기
- ✅ 한글로 출력하기
- ✅ 코드 실행하기

---

## 📦 1단계: K-FreeLang 설치

### macOS / Linux
```bash
npm install -g kfreelang
```

### Windows
```bash
npm install -g kfreelang
```

### 설치 확인
```bash
kfreelang --version
```

---

## 📝 2단계: 첫 번째 코드 작성

### 파일 생성
`hello.free` 파일을 만듭니다:

```freelang
출력("안녕하세요, 프리랭!")
```

### 코드 실행
```bash
kfreelang hello.free
```

### 결과
```
안녕하세요, 프리랭!
```

## ✏️ 연습 문제

### 문제 1: 자신의 이름 출력하기
다음과 같이 출력하는 코드를 작성하세요:
```
안녕하세요!
저는 [당신의 이름]입니다.
프리랭을 배우고 있습니다.
```

**힌트**: `출력()` 함수를 여러 번 사용하세요.

### 문제 2: 숫자 계산해서 출력하기
다음과 같이 출력하는 코드를 작성하세요:
```
10 + 20 = 30
100 - 30 = 70
5 * 6 = 30
100 / 5 = 20
```

**힌트**: 계산 결과를 직접 `출력()` 함수의 인자로 넣으세요.

### 문제 3: 한글 문법 연습
다음을 모두 출력하는 코드를 작성하세요:
```
프리랭 특징:
1. 완벽한 한글 지원
2. 594개 함수
3. 46개 모듈
4. 자가호스팅
```

**힌트**: 숫자는 그냥 숫자 그대로 출력하면 됩니다.

## 🔑 핵심 개념

### 출력 함수 (출력)
```freelang
출력(값)
```
- 주어진 값을 화면에 출력합니다
- 문자열, 숫자, 계산식 등을 출력할 수 있습니다

### 주석
```freelang
// 이것은 한 줄 주석입니다
/* 이것은 여러 줄 주석입니다 */
```
- 주석은 코드 실행에 포함되지 않습니다
- 코드 설명을 위해 사용합니다

### 문자열
```freelang
"이것은 문자열입니다"
'작은 따옴표도 사용 가능합니다'
```
- 따옴표로 감싼 텍스트입니다
- 숫자와 다릅니다

## 💡 팁

### 코드 작성 팁
1. 파일 이름은 영문으로 지으세요 (예: `hello.free`)
2. `.free` 확장자를 사용하세요

### 문제 해결
**"kfreelang: command not found"**
- npm이 제대로 설치되었는지 확인하세요
- `npm install -g kfreelang`을 다시 실행하세요

**파일을 찾을 수 없습니다**
- 파일이 현재 폴더에 있는지 확인하세요
- `ls` 명령어로 파일 목록을 확인하세요
