# Step 🔟: 실전 프로젝트

**난이도**: ⭐⭐⭐⭐ 고급 | **소요 시간**: 60분

## 📋 프로젝트 목록

### 프로젝트 1: 계산기 (20분)

**요구사항**:
- 두 수의 산술 연산 (+, -, ×, ÷)
- 입력값 검증
- 에러 처리

**핵심 개념 복습**:
- 함수
- 조건문
- 에러 처리

### 프로젝트 2: TODO 앱 (25분)

**요구사항**:
- 할일 추가
- 할일 완료 표시
- 할일 삭제
- 전체 목록 출력

**핵심 개념 복습**:
- 배열
- 구조체
- 반복문

### 프로젝트 3: 성적 관리 시스템 (15분)

**요구사항**:
- 학생 정보 저장 (이름, 학번, 점수)
- 성적 통계 (평균, 최고, 최저)
- 학생 검색

**핵심 개념 복습**:
- 구조체
- 배열
- 함수

## 🤝 커뮤니티 활동

### 버그 보고하기
GitHub Issues에서 발견한 버그를 보고하세요.

```
Title: [Bug] 계산기가 음수를 처리하지 못함
Description:
- 현재 상황: -5 + 3을 입력하면 에러 발생
- 예상 상황: -2가 반환되어야 함
```

### 기능 제안하기
새로운 기능 아이디어를 GitHub Discussions에서 제안하세요.

```
Title: [Feature] 과학 계산기 모드
Description:
- 삼각함수 (sin, cos, tan)
- 지수 및 로그
- 상수 (π, e)
```

### 프로젝트 공유하기
당신이 만든 프로젝트를 커뮤니티와 공유하세요!

```
Title: [Project] 음악 재생 목록 관리 앱
Description:
- 곡 추가, 삭제, 재생
- 플레이리스트 저장
- GitHub: [링크]
```

### 좋은 코드 작성 팁
1. **명확한 변수명 사용**: `a`, `b`보다는 `점수`, `평균`
2. **주석 남기기**: 복잡한 로직은 설명 추가
3. **함수로 분리하기**: 같은 코드는 함수로 만들기
4. **테스트하기**: 엣지 케이스도 테스트
