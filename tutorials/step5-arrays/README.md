## 📚 핵심 개념

### 배열 생성
```freelang
변수 숫자 = [1, 2, 3, 4, 5]
변수 문자열 = ["사과", "바나나", "딸기"]
변수 혼합 = [1, "문자", 3.14]
```

### 배열 접근
```freelang
배열[0]           // 첫 번째 요소
배열[배열.길이 - 1]  // 마지막 요소
```

### 배열 메서드
- `push(요소)`: 끝에 요소 추가
- `pop()`: 끝에서 요소 제거
- `shift()`: 앞에서 요소 제거
- `unshift(요소)`: 앞에 요소 추가
- `filter(함수)`: 조건에 맞는 요소만
- `map(함수)`: 각 요소를 변환
- `join(구분자)`: 배열을 문자열로

## 📝 예제 코드

### 예제 1: 기본 배열
파일: `1-array-basics.free`

```freelang
변수 과일 = ["사과", "바나나", "딸기"]

// 배열 요소 접근
출력(과일[0])
출력(과일[1])
출력(과일[2])

// 배열 길이
출력(과일.길이)

// 배열 수정
과일[1] = "포도"
출력(과일[1])
```

### 예제 2: 배열 메서드
파일: `2-array-methods.free`

```freelang
변수 숫자 = [1, 2, 3]

// push: 끝에 추가
숫자.push(4)
출력(숫자)

// filter: 조건 필터링
변수 짝수 = 숫자.filter(함수(x) {
  반환 x % 2 == 0
})

// map: 변환
변수 두배 = 숫자.map(함수(x) {
  반환 x * 2
})
```

### 예제 3: 다차원 배열
파일: `3-2d-arrays.free`

```freelang
변수 표 = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

출력(표[0][0])  // 1
출력(표[1][1])  // 5
출력(표[2][2])  // 9
```

## 💾 솔루션

### 문제 1 솔루션
```freelang
함수 배열_합(배열) {
  변수 합 = 0
  반복 (변수 i = 0; i < 배열.길이; i += 1) {
    합 = 합 + 배열[i]
  }
  반환 합
}
```

### 문제 2 솔루션
```freelang
함수 배열_뒤집기(배열) {
  변수 역순 = []
  반복 (변수 i = 배열.길이 - 1; i >= 0; i -= 1) {
    역순.push(배열[i])
  }
  반환 역순
}
```

### 문제 3 솔루션
```freelang
함수 배열_포함(배열, 값) {
  반복 (변수 i = 0; i < 배열.길이; i += 1) {
    만약 배열[i] == 값 {
      반환 참
    }
  }
  반환 거짓
}
```
