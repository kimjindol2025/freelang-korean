## 📚 핵심 개념

### 변수 (Variable)
값을 저장하고 재사용할 수 있는 공간입니다.

```freelang
변수 이름 = "김철수"
변수 나이 = 25
```

### 상수 (Constant)
한 번 정의한 후 변경할 수 없는 값입니다.

```freelang
상수 PI = 3.14159
상수 최대_학생_수 = 30
```

### 자료형 (Data Type)
- **숫자**: `10`, `3.14`, `-5`
- **문자열**: `"안녕하세요"`, `'Hello'`
- **참/거짓**: `참`, `거짓`

### 연산자 (Operators)

**산술 연산자**:
```freelang
10 + 5      // 덧셈 (15)
10 - 3      // 뺄셈 (7)
4 * 5       // 곱셈 (20)
20 / 4      // 나눗셈 (5)
10 % 3      // 나머지 (1)
```

**비교 연산자**:
```freelang
10 > 5      // 크다 (참)
10 < 5      // 작다 (거짓)
10 >= 10    // 크거나 같다 (참)
10 == 10    // 같다 (참)
10 != 5     // 다르다 (참)
```

**논리 연산자**:
```freelang
참이고 참    // 그리고 (AND) - 참
참이고 거짓  // 그리고 (AND) - 거짓
참또는 거짓  // 또는 (OR) - 참
```

## 📝 예제 코드

### 예제 1: 기본 변수 선언
파일: `1-variables.free`

```freelang
// 변수 선언과 초기화
```

### 예제 2: 자료형 이해하기
파일: `2-types.free`

```freelang
// 숫자형 변수
변수 정수 = 42
변수 실수 = 3.14

// 문자열 변수
변수 문자열1 = "안녕하세요"
변수 문자열2 = '프리랭입니다'

// 논리형 변수
변수 참값 = 참
변수 거짓값 = 거짓

// 각 타입 출력
출력("=== 숫자형 ===")
출력(정수)
출력(실수)

출력("=== 문자열 ===")
출력(문자열1)
출력(문자열2)

출력("=== 논리형 ===")
출력(참값)
출력(거짓값)
```

### 예제 3: 연산자 활용
파일: `3-operators.free`

```freelang
// 산술 연산자
변수 a = 10
변수 b = 3

출력("=== 산술 연산 ===")
출력(a + b)  // 13
출력(a - b)  // 7
출력(a * b)  // 30
출력(a / b)  // 3
출력(a % b)  // 1

// 비교 연산자
출력("=== 비교 연산 ===")
출력(a > b)   // 참
출력(a < b)   // 거짓
출력(a == b)  // 거짓
출력(a != b)  // 참

// 논리 연산자
출력("=== 논리 연산 ===")
변수 조건1 = 참
변수 조건2 = 거짓

출력(조건1이고 조건2)  // 거짓
출력(조건1또는 조건2)  // 참
```

## ✏️ 연습 문제

### 문제 1: 사각형의 넓이 계산
다음을 계산하고 출력하는 코드를 작성하세요:
- 가로: 15cm, 세로: 20cm인 사각형의 넓이
- 가로: 10cm, 세로: 10cm인 정사각형의 넓이

```
사각형 넓이: 300
정사각형 넓이: 100
```

**힌트**: 넓이 = 가로 × 세로

### 문제 2: 온도 변환
섭씨 온도를 화씨 온도로 변환하는 코드를 작성하세요:
- 섭씨 25도는 화씨로 몇 도인가?
- 섭씨 0도는 화씨로 몇 도인가?

```
25°C = 77.0°F
0°C = 32.0°F
```

**힌트**: 화씨 = (섭씨 × 9/5) + 32

### 문제 3: 논리 연산 이해
다음 조건을 확인하는 코드를 작성하세요:
- 나이가 20살 이상인가?
- 이름이 "김철수"인가?
- 둘 다 만족하는가?

```
나이 >= 20: 참/거짓
이름 == "김철수": 참/거짓
둘 다 만족: 참/거짓
```

**힌트**: "이고" 연산자를 사용하세요.

## 🔑 중요한 규칙

### 변수명 작성 규칙
- 한글, 영문, 숫자 혼용 가능
- 첫 글자는 한글 또는 영문
- 예: `학생_이름`, `student_name`, `나이`, `age`

### 문자열 표현
- 큰 따옴표로 감싸기: `"안녕하세요"`
- 작은 따옴표도 가능: `'Hello'`
- 문자열 연결: `"안녕" + "하세요"` → `"안녕하세요"`

### 타입 확인
```freelang
변수 x = 10      // 숫자
변수 y = "10"    // 문자열
출력(x + 5)      // 15 (숫자 연산)
출력(y + "5")    // "105" (문자열 연결)
```
