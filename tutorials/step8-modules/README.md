# Step 8️⃣: 모듈과 라이브러리

**난이도**: ⭐⭐⭐ 중급 | **소요 시간**: 40분

> 모듈을 통해 코드를 조직하고 라이브러리를 활용합니다.
> 이 Step을 완료하면 다른 개발자의 코드를 재사용할 수 있게 됩니다.

---

## 🎯 학습 목표

- ✅ 모듈 import/export 이해하기
- ✅ 표준 라이브러리 사용하기
- ✅ 외부 패키지 활용하기
- ✅ 코드 재사용성 높이기

---

## 📚 핵심 개념

### 모듈 임포트
```freelang
임포트 "수학모듈"
임포트 "문자열처리"
```

### K-StdLib 사용
```freelang
// Essential Libraries
문자열_concat("안녕", "하세요")
배열_필터([1, 2, 3], 필터함수)
수학_덧셈(10, 20)
JSON_stringify(객체)
```

### 표준 라이브러리 예제
```freelang
변수 합 = 수학_덧셈(5, 3)
변수 정렬된배열 = 배열_정렬([3, 1, 2])
변수 json문자열 = JSON_stringify({이름: "kim"})
```

---

## 📝 예제

### 예제 1: 모듈 기본
```freelang
임포트 "표준라이브러리"

변수 결과 = 문자열_길이("프리랭")
출력(결과)
```

### 예제 2: 여러 모듈
```freelang
임포트 "수학"
임포트 "배열"
임포트 "문자열"

변수 합 = 수학_덧셈(10, 20)
변수 정렬 = 배열_정렬([3, 1, 2])
```

### 예제 3: K-StdLib 활용
```freelang
변수 json_data = JSON_stringify({
  이름: "김철수",
  나이: 25
})
```
