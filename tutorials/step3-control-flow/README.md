## 📚 핵심 개념

### if 조건문
조건이 참일 때만 코드를 실행합니다.

```freelang
만약 조건 {
  // 조건이 참일 때 실행
}
```

### if-else 조건문
조건이 참이면 if 블록, 거짓이면 else 블록을 실행합니다.

```freelang
만약 조건 {
  // 참일 때
} 아니면 {
  // 거짓일 때
}
```

### for 반복문
정해진 횟수만큼 반복합니다.

```freelang
반복 (변수 = 0; 변수 < 10; 변수 += 1) {
  // 10번 반복
}
```

### while 반복문
조건이 참인 동안 계속 반복합니다.

```freelang
반복 {
  // 조건이 참일 때 계속 실행
  만약 조건 {
    나가  // 반복 탈출
  }
}
```

### break와 continue
- **break**: 반복문을 즉시 종료
- **continue**: 현재 반복을 건너뛰고 다음 반복으로

## 📝 예제 코드

### 예제 1: if-else 조건문
파일: `1-if-else.free`

```freelang
// 성적 판정 프로그램
변수 점수 = 85

만약 점수 >= 90 {
  출력("A 학점")
} 아니면 {
  만약 점수 >= 80 {
    출력("B 학점")
  } 아니면 {
    만약 점수 >= 70 {
      출력("C 학점")
    } 아니면 {
      출력("F 학점")
    }
  }
}
```

### 예제 2: for 반복문
파일: `2-for-loop.free`

```freelang
// 1부터 5까지 출력
반복 (변수 i = 1; i <= 5; i += 1) {
  출력(i)
}

// 2의 배수 찾기
출력("=== 2의 배수 ===")
반복 (변수 j = 1; j <= 10; j += 1) {
  만약 j % 2 == 0 {
    출력(j)
  }
}
```

### 예제 3: while 반복문
파일: `3-while-loop.free`

```freelang
// 구구단 2단 출력
변수 i = 1

반복 {
  만약 i > 9 {
    나가
  }

  변수 결과 = 2 * i
  출력("2 × " + i + " = " + 결과)

  i += 1
}
```

## 🔑 중요한 규칙

### 조건문의 중첩
```freelang
만약 조건1 {
  만약 조건2 {
    // 두 조건 모두 참일 때 실행
  }
}
```

### 반복문의 break와 continue
```freelang
반복 (변수 i = 1; i <= 10; i += 1) {
  만약 i == 3 {
    계속      // i가 3일 때 건너뛰기
  }

  만약 i == 7 {
    나가      // i가 7일 때 반복 종료
  }

  출력(i)
}
```

## 💡 팁

### 무한 반복 피하기
```freelang
// ❌ 위험: 무한 반복!
반복 (변수 i = 0; i < 10; i += 1) {
  // i를 변경하지 않음
}

// ✅ 안전: 증가시킴
반복 (변수 i = 0; i < 10; i += 1) {
  i += 1
}
```
