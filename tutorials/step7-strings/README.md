# Step 7️⃣: 문자열과 정규식

**난이도**: ⭐⭐⭐ 중급 | **소요 시간**: 45분

> 문자열을 조작하고 패턴 매칭을 통해 텍스트를 처리합니다.
> 이 Step을 완료하면 복잡한 텍스트 데이터를 효율적으로 다룰 수 있게 됩니다.

## 📚 핵심 개념

### 문자열 메서드
```freelang
"프리랭".길이           // 3
"프리랭".포함("랭")      // 참
"프리랭".시작("프")      // 참
"프리랭".끝("랭")       // 참
"프리랭".대문자()        // 영어일 때만
"프리랭".소문자()        // 영어일 때만
```

### 문자열 템플릿
```freelang
변수 이름 = "프리랭"
변수 메시지 = "안녕, " + 이름 + "!"
```

### 정규식 기본
```
이메일: .*@.*\..*
전화번호: [0-9]{2,3}-[0-9]{3,4}-[0-9]{4}
```

## 📝 예제

### 예제 1: 문자열 메서드
```freelang
변수 텍스트 = "Hello World"

출력(텍스트.길이)
출력(텍스트.포함("World"))
출력(텍스트.시작("Hello"))
```

### 예제 2: 문자열 변환
```freelang
변수 이름 = "kim"
변수 메시지 = "안녕하세요, " + 이름 + "님!"
출력(메시지)
```

### 예제 3: 정규식 검증
```freelang
함수 이메일_검증(이메일) {
  // 간단한 이메일 패턴 검사
  반환 이메일.포함("@")이고 이메일.포함(".")
}
```
