## 빌트인 함수

### 출력
**용도**: 콘솔에 값 출력

```freelang
함수 출력(값: 모든타입) → 공집합 {
  // JavaScript: console.log()와 동일
}
```

**사용 예**:
```freelang
출력("Hello")           // Hello
출력(42)               // 42
출력([1, 2, 3])        // [1, 2, 3]
```

### 타입
**용도**: 값의 타입 반환

```freelang
함수 타입(값: 모든타입) → 문자열 {
  // 반환: 'number', 'string', 'boolean', 'array', 'object', 'null'
}
```

**사용 예**:
```freelang
타입(42)              // "number"
타입("hello")         // "string"
타입([1, 2])          // "array"
```

### 길이
**용도**: 배열, 문자열 길이 반환

```freelang
변수 문자열 = "hello"
출력(문자열.len)      // 5

변수 배열 = [1, 2, 3]
출력(배열.len)        // 3
```

## 배열 메서드

### 기본 메서드

| 메서드 | 한글 | 설명 | 반환값 |
|--------|------|------|--------|
| `push(값)` | - | 끝에 요소 추가 | 새 길이 |
| `pop()` | - | 마지막 요소 제거 | 제거된 요소 |
| `shift()` | - | 첫 요소 제거 | 제거된 요소 |
| `unshift(값)` | - | 앞에 요소 추가 | 새 길이 |
| `concat(배열)` | 합치기 | 배열 합치기 | 새 배열 |
| `slice(시작, 끝)` | - | 부분 추출 | 새 배열 |
| `length` / `len` | 길이 | 배열 길이 | 숫자 |

**사용 예**:
```freelang
변수 배열 = [1, 2, 3]

배열.push(4)          // [1, 2, 3, 4]
배열.pop()            // [1, 2, 3]
배열.concat([4, 5])   // [1, 2, 3, 4, 5]

출력(배열.len)        // 3
```

### 검색 메서드

| 메서드 | 설명 |
|--------|------|
| `includes(값)` | 포함 여부 반환 |
| `indexOf(값)` | 위치 반환 (-1 없음) |
| `find(함수)` | 조건 만족 첫 요소 |
| `filter(함수)` | 조건 만족 모든 요소 |
| `map(함수)` | 변환된 배열 반환 |

**사용 예**:
```freelang
변수 배열 = [1, 2, 3, 4, 5]

배열.includes(3)      // 참
배열.indexOf(3)       // 2
```

## 문자열 메서드

### 변환 메서드

| 메서드 | 한글 | 설명 |
|--------|------|------|
| `toUpperCase()` | 대문자 | 대문자 변환 |
| `toLowerCase()` | 소문자 | 소문자 변환 |
| `trim()` | 공백제거 | 앞뒤 공백 제거 |
| `repeat(숫자)` | 반복 | N번 반복 |

**사용 예**:
```freelang
변수 문자 = "Hello World"

출력(문자.toUpperCase())   // HELLO WORLD
출력(문자.toLowerCase())   // hello world
출력("ha".repeat(3))       // hahaha
```

### 검색 메서드

| 메서드 | 설명 |
|--------|------|
| `includes(문자)` | 포함 여부 |
| `startsWith(문자)` | 시작 여부 |
| `endsWith(문자)` | 끝 여부 |
| `indexOf(문자)` | 위치 검색 |

**사용 예**:
```freelang
변수 문자 = "hello world"

문자.includes("world")     // 참
문자.startsWith("hello")   // 참
문자.endsWith("world")     // 참
```

### 분할/합치기

| 메서드 | 설명 |
|--------|------|
| `split(구분)` | 배열로 분할 |
| `substring(시작, 끝)` | 부분 추출 |
| `replace(찾기, 바꾸기)` | 첫 매칭 변경 |

**사용 예**:
```freelang
변수 문자 = "a,b,c"

문자.split(",")        // ["a", "b", "c"]
문자.substring(0, 1)   // "a"
```

## 암호화 (Crypto)

### OTP (일회용 암호)

```freelang
// TOTP 생성 (시간 기반)
변수 코드 = TOTP_생성("비밀키")
// 반환: "123456" (6자리)

// TOTP 검증
변수 유효 = TOTP_검증("비밀키", "123456")
// 반환: 참 또는 거짓

// HOTP 생성 (카운터 기반)
변수 코드 = HOTP_생성("비밀키", 1)

// HOTP 검증
변수 유효 = HOTP_검증("비밀키", "123456", 1)
```

### 인코딩

```freelang
// Base32 인코딩
변수 인코딩 = Base32_인코딩("hello")

// Base32 디코딩
변수 디코딩 = Base32_디코딩(인코딩)

// Base64 인코딩
변수 b64 = Base64_인코딩("hello")

// Hex 인코딩
변수 hex = Hex_인코딩("hello")
```

### 해싱

```freelang
// SHA-256 해시
변수 해시 = SHA256("데이터")
// 반환: "abcd...ef" (16진수, 64자)

// HMAC-SHA1
변수 서명 = HMAC_SHA1("비밀키", "데이터")
```

## 규제 준수 (Compliance)

### 한국 번호 검증

```freelang
// 주민등록번호 검증
만약 RRN_유효한가("123456-1234567") {
  출력("유효한 주민등록번호")
}

// 나이 추출
변수 나이 = RRN_나이_추출("950101-1234567")
// 반환: 30 (2025년 기준)

// 사업자등록번호 검증
만약 BRN_유효한가("123-45-67890") {
  출력("유효한 사업자등록번호")
}
```

### 전화번호 검증

```freelang
// 한국 전화번호 검증
만약 한국전화번호_유효한가("010-1234-5678") {
  출력("유효한 전화번호")
}

// 이메일 검증
만약 이메일_유효한가("user@example.com") {
  출력("유효한 이메일")
}
```

### 개인정보 마스킹 (PIPA)

```freelang
// 이름 마스킹
변수 마스킹 = 이름_마스킹("김철수")
// 반환: "김*수"

// 전화번호 마스킹
변수 마스킹 = 전화번호_마스킹("010-1234-5678")
// 반환: "010-****-5678"

// 이메일 마스킹
변수 마스킹 = 이메일_마스킹("user@example.com")
// 반환: "u***@example.com"
```

## 🎯 빠른 시작

### 예제 1: 배열 처리
```freelang
변수 배열 = [1, 2, 3, 4, 5]

// 길이 확인
출력(배열.len)           // 5

// 조건 필터
변수 짝수 = 배열.filter(함수(x) {
  반환 x % 2 == 0
})
출력(짝수)               // [2, 4]

// 변환
변수 제곱 = 배열.map(함수(x) {
  반환 x * x
})
출력(제곱)               // [1, 4, 9, 16, 25]
```

### 예제 2: 개인정보 처리
```freelang
// 사용자 정보
변수 이름 = "김철수"
변수 전화 = "010-1234-5678"
변수 이메일 = "kim@example.com"

// 마스킹
변수 마스킹이름 = 이름_마스킹(이름)
변수 마스킹전화 = 전화번호_마스킹(전화)
변수 마스킹이메일 = 이메일_마스킹(이메일)

// 출력
출력("이름: " + 마스킹이름)           // 이름: 김*수
출력("전화: " + 마스킹전화)           // 전화: 010-****-5678
출력("이메일: " + 마스킹이메일)       // 이메일: u***@example.com
```

### 예제 3: 2FA (2단계 인증)
```freelang
// 비밀키 생성
변수 비밀키 = 비밀키_생성()

// TOTP 코드 생성
변수 코드 = TOTP_생성(비밀키)
출력("현재 코드: " + 코드)

// 사용자 입력 검증
변수 사용자코드 = "123456"
만약 TOTP_검증(비밀키, 사용자코드) {
  출력("✅ 인증 성공")
} 아니면 {
  출력("❌ 인증 실패")
}
```
