FreeLang

고성능 프로그래밍 언어. Native 라이브러리, 현대적인 컴파일러, 풍부한 도구 생태계

GitHub

강력한 기능

⚡

고성능

최적화된 컴파일러와 런타임으로 빠른 실행 속도를 제공합니다. Zero-overhead abstraction.

🔧

Native 라이브러리

JWT 인증, ORM, HTTP 클라이언트 등 6개의 프로덕션 레벨 Native 라이브러리를 제공합니다.

📚

1,500+ 빌트인 함수

문자열, 배열, 날짜, 수학 등 다양한 카테고리의 풍부한 표준 라이브러리.

🛠️

완전한 도구 생태계

CLI, LSP (IDE 지원), REPL, 디버거, 테스트 러너 등을 포함합니다.

📦

npm 패키지 지원

npm registry와 완전히 호환되는 패키지 시스템으로 손쉬운 의존성 관리.

🎯

현대적인 문법

함수형, 객체지향, 절차형 패러다임을 모두 지원하는 유연한 문법.

시작하기

1️⃣ FreeLang 컴파일러 설치

npm install -g @freelang/cli

2️⃣ Native 라이브러리 설치

npm install @freelang/auth @freelang/validator @freelang/orm

3️⃣ 첫 프로그램 작성

cat > hello.fl << 'EOF' fn main() { print("Hello, FreeLang!") } main() EOF freelang hello.fl

JWT HMAC Zero-deps
@freelang/validator

입력 검증

스키마 기반 검증과 커스텀 규칙을 지원하는 강력한 검증 라이브러리.

Validation Schema Zero-deps
@freelang/security

보안 미들웨어

CORS, CSP, Rate-limiting, 보안 헤더 등 웹 보안 기능 제공.

CORS CSP Rate-limiting
@freelang/orm

데이터베이스 ORM

Sequelize 대체. 쿼리 빌더, 관계, 트랜잭션, 연결 풀링 지원.

ORM Query Builder Transactions
@freelang/web-forge

SSG/SSR 웹 프레임워크

21+ 함수를 포함한 강력한 웹 렌더링 엔진. 템플릿, 컴포넌트 지원.

SSG SSR Templates
@freelang/http

HTTP 클라이언트
