🎯 가장 간단한 프로그램

한국형 프로그래밍 언어로 첫 번째 프로그램을 작성해봅시다.

출력("안녕하세요, K-FreeLang!")
📌 개념 설명:
• 출력() - 내장 함수로 값을 출력합니다
• "..." - 문자열 리터럴
• 한글로 코드를 작성합니다!

📝 변수 사용

변수 이름 = "K-FreeLang" 출력("환영합니다, " + 이름 + "!")
📌 주요 개념:
• 변수 - 변수 선언 (let)
• = - 할당 연산자
• + - 문자열 연결

🔧 함수 정의

함수 인사(이름: 문자열) -> 공집합 { 출력("안녕하세요, " + 이름 + "!") } 인사("K-FreeLang") 인사("한국")
📌 주요 개념:
• 함수 - 함수 정의 (fn)
• (이름: 문자열) - 파라미터 + 타입
• -> 공집합 - 반환 타입
• {} - 함수 본체

⚡ 조건문

변수 나이 = 25 만약 나이 >= 20 { 출력("성인입니다") } 아니면 { 출력("미성년자입니다") }
📌 주요 개념:
• 만약 - 조건문 (if)
• >= - 비교 연산자
• 아니면 - else

🔁 반복문

반복 i = 1; i <= 5; i = i + 1 { 출력(i) }

출력:

1 2 3 4 5
📌 주요 개념:
• 반복 - for 루프
• 초기화; 조건; 업데이트 - C 스타일 for
💡 팁: 완전히 한글로 프로그래밍하기
K-FreeLang은 완전히 한글로 프로그래밍할 수 있습니다:
함수 우리말계산(첫번째: 숫자, 두번째: 숫자) -> 숫자 { 반환 첫번째 + 두번째 } 변수 결과 = 우리말계산(10, 20) 출력(결과) // 30
• 변수명, 함수명도 한글 가능
• 영문과 한글 혼합 가능
• 주석도 자유롭게 한글 사용 가능

2️⃣ K-StdLib 사용법

중급

난이도: 중급 | 소요시간: 20분

선수 지식: 기본 문법, 함수, 배열, 객체

🎯 목표

K-FreeLang 표준 라이브러리(K-StdLib)의 핵심 모듈을 사용하여 실제 프로그램을 작성합니다.

📚 핵심 모듈:
• math - 수학 연산
• array - 배열 조작
• string - 문자열 처리
• object - 객체 조작
• io - 파일 I/O

📦 모듈 사용 선언

K-FreeLang에서 외부 모듈을 사용하려면 사용 키워드를 사용합니다:

// 표준 라이브러리 모듈 로드 사용 math 사용 array 사용 string 사용 object 사용 io

🔢 math 모듈 - 수학 연산

기본 함수

사용 math // 절대값 변수 abs1 = 절대값(-5) // 5 변수 abs2 = 절대값(3.14) // 3.14 // 최대/최소값 변수 max = 최대값(10, 20) // 20 변수 min = 최소값(10, 20) // 10 // 거듭제곱 변수 제곱 = 거듭제곱(2, 8) // 256 변수 세제곱 = 거듭제곱(3, 3) // 27

실전 예제: 통계 계산

사용 math 사용 array 함수 평균(숫자들: 배열<숫자>) → 숫자 { 변수 합 = 합계(숫자들) 변수 개수 = 길이(숫자들) 반환 합 / 개수 } // 사용 변수 데이터 = [85, 90, 78, 92, 88] 출력(평균(데이터)) // 86.6

📋 array 모듈 - 배열 조작

사용 array 변수 숫자들 = [1, 2, 3, 4, 5] // 배열 길이 변수 크기 = 길이(숫자들) // 5 // 배열 수정 변수 추가됨 = [타입(숫자들), 6] // [1,2,3,4,5,6] 변수 제거됨 = 슬라이스(숫자들, 0, 4) // [1,2,3,4] // 배열 합계 변수 합계 = 합계(숫자들) // 15

🔤 string 모듈 - 문자열 처리

사용 string 변수 텍스트 = "K-FreeLang" // 길이 변수 길이 = 길이(텍스트) // 11 // 대소문자 변환 변수 대문자 = 대문자(텍스트) // "K-FREELANG" 변수 소문자 = 소문자(텍스트) // "k-freelang" // 문자열 분할 변수 단어들 = 분할(텍스트, "-") // ["K", "FreeLang"] // 문자열 합치기 변수 합쳐진 = 결합(단어들, " ") // "K FreeLang"

📁 io 모듈 - 파일 I/O

사용 io // 파일 읽기 변수 내용 = 파일읽기("data.txt") // 파일 쓰기 파일쓰기("output.txt", "K-FreeLang은 훌륭합니다") // 파일 추가 파일추가("log.txt", "\n새로운 로그 항목") // 파일 존재 확인 만약 파일존재("data.txt") { 출력("파일이 존재합니다") }

실전 예제: 간단한 로깅

사용 io 함수 로그기록(메시지: 문자열) -> 공집합 { 변수 타임스탬프 = 현재날짜시간() 변수 로그항목 = "[" + 타임스탬프 + "] " + 메시지 파일추가("app.log", 로그항목 + "\n") } // 사용 로그기록("애플리케이션 시작됨") 로그기록("데이터 처리 완료") 로그기록("애플리케이션 종료됨")

3️⃣ 통합 예제: 학생 성적 관리 시스템

고급

난이도: 고급 | 소요시간: 30분

🎓 프로젝트 목표

학생들의 성적을 입력받아 평균을 계산하고 파일에 저장하는 시스템을 만들어봅시다.

📝 완전한 예제 코드

사용 math 사용 array 사용 object 사용 string 사용 io 함수 성적계산(점수들: 배열<숫자>) → 맵<문자열, 숫자> { 변수 합 = 0 반복 변수 i = 0; i < 길이(점수들); i = i + 1 { 합 = 합 + 점수들[i] } 변수 평균 = 합 / 길이(점수들) 변수 최고 = 최대값(점수들[0], 점수들[1]) 변수 최저 = 최소값(점수들[0], 점수들[1]) 반환 { 평균: 평균, 최고점: 최고, 최저점: 최저 } } 함수 학생정보_저장(학생: 맵<문자열, 숫자>) -> 공집합 { 변수 CSV = "이름," + 학생["이름"] CSV = CSV + "\n평균점수," + 학생["평균"] 파일쓰기("student_" + 학생["학번"] + ".txt", CSV) } // 메인 프로그램 변수 학생들 = [ { 이름: "김프리랭", 학번: "20240001", 수학: 95, 영어: 87, 국어: 92 }, { 이름: "이독립언어", 학번: "20240002", 수학: 88, 영어: 90, 국어: 85 } ] 반복 변수 i = 0; i < 길이(학생들); i = i + 1 { 변수 학생 = 학생들[i] 변수 점수들 = [학생["수학"], 학생["영어"], 학생["국어"]] 변수 성적 = 성적계산(점수들) 출력("학생: " + 학생["이름"]) 출력("평균: " + 성적["평균"]) }
